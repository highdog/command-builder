蓝牙协议示例

RFCOMM 协议格式 (以 GetBatteryLevel 为例)

FF0400002E502206
SOF Ver Flags Len Vendor ID Feature+Pkt+Cmd

RFCOMM 协议使用完整的帧格式，包含帧头、版本、标志、长度等完整信息。

GATT over BLE 协议格式 (以 GetBatteryLevel 为例)

2E502206
Vendor ID Feature+Pkt+Cmd

GATT over BLE 协议使用较简洁的格式，直接包含厂商ID和命令信息。

字段说明

  • SOF: Start of Frame (帧开始标志)
  • Ver: Version (协议版本)
  • Flags: 标志位
  • Len: Length (数据长度)
  • Vendor ID: 厂商标识符
  • Feature+Pkt+Cmd: 功能+包类型+命令组合
